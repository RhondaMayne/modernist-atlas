# Books

The Books section contains the primary-text reading programme for *The Modernist Atlas*. It is organised around monthly themes that build a structured understanding of literary modernism.

Each completed work has its own note in the `books/` folder. These notes examine the work's modernist significance, formal innovations, key concepts, intellectual connections, and the permanent notes generated through reading.
